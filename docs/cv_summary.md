# CV Summary

**Cyber Log Text Classifier ML**

Built a machine-learning pipeline that classifies security logs into attack categories such as SQL Injection, XSS, brute-force attempts, port scans, path traversal, malware indicators, authentication errors, and benign events.

## Technologies

Python, pandas, scikit-learn, TF-IDF, Logistic Regression, joblib, pytest

## Security Value

The project demonstrates how ML can support SOC triage by quickly grouping suspicious log messages into likely attack classes. It is educational and can be extended with real SIEM logs, normalization, enrichment, and deployment as an API.
